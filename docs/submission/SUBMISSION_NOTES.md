# Ride, Revere, Ride! Submission Notes

## Project Title

Ride, Revere, Ride!

## Theme Connection

Built for Summer into AI Theme 1: 8-Bit America. The game turns a Revolutionary War folk-history image into a playful retro score-attack ride: wake the town before dawn by delivering warning broadsides while avoiding colonial-era parody hazards.

## Short Pitch

Ride through a moonlit colonial route, throw broadsides, dodge loyalist hazards, and wake the town before dawn.

## Gameplay Summary

The player controls Revere across four lanes. Targets and hazards scroll from right to left. Throw broadsides at buildings to score, build combo multipliers, and raise the alarm meter. Avoid hazards to preserve stamina. The run ends when dawn arrives or stamina reaches zero.

## AI Tools Used

- ChatGPT: game concept, design direction, art direction, and prompt/reference generation.
- Codex: implementation, integration, cleanup, verification, and review packaging.
- Phaser, Vite, and TypeScript: browser game framework and build stack.

## What Was Built

- One-level 90-second score-attack loop.
- Title, gameplay, game-over, and high-score scenes.
- Local-only high scores.
- Wave-based fair spawn director.
- Combo multiplier scoring and alarm meter milestones.
- Prototype AI-assisted runtime art derived from references.
- Generated WebAudio cues with mute toggle.
- GitHub Pages deployment workflow.

## What Remains Prototype

- Runtime sprites are prototype derivatives, not final commercial production art.
- Sprite frame animation is minimal; most motion uses transforms and small Phaser effects.
- Audio is generated in code and not a final composed soundtrack or sound-design pass.
- Balance is tuned for a public demo, not final arcade mastery.
