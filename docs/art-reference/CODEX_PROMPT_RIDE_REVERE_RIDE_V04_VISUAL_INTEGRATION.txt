Context:
The current v0.3 prototype is already playable with placeholder rectangles, a 90-second run, four lanes, score, combo multiplier, alarm meter, stamina, hazards, targets, game-over initials, local high scores, and GitHub Pages workflow. Primary visual direction:
- Lost 16-bit/SNES-style arcade cartridge.
- Moonlit colonial road.
- Parchment, wood, iron, lantern glow, muted midnight blues, warm amber windows, Crown red accents, Revolutionary blue accents.
- Chunky readable pixels, not strict 8x8 NES minimalism.
- Nearest-neighbor scaling everywhere.
- Funny historical parody, but readable first.
- No direct Paperboy references, names, or assets.

Important game design state rules for target buildings:
- Unwarned state: mostly dark, quiet, unaware. The porch/front lantern may be on, but windows should mostly be dark.
- Warned/alert state: windows light up, posted broadsides/signs appear, lanterns brighten, citizens/patriots implied awake.
- House: porch lantern on before warning; after warning, many/all windows lit and a warning broadside posted.
- Tavern: dim/quiet before; after warning, warm/lively, windows bright, broadsides on wall/door.
- Church: dim/quiet before; after warning, bell tower/windows lit and broadside posted.
- Blacksmith: forge dim before; after warning, forge is much brighter, shop glows orange, signs/broadsides posted.
- Militia Post: quiet before; after warning, lanterns bright, militia active. Sign should read: `LIBERTY OR DEATH MILITIA`.

Historical/funny Easter egg:
- Hemp farm scenery should be a working colonial hemp farm, not a modern weed gag.
- Include one subtle hemp leaf stamp on a bale/crate only. No smoke jokes, no 420 jokes.
